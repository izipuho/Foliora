# Catalog

Нативное iOS-приложение на SwiftUI для каталогизации домашних коллекций.

Сейчас в проекте уже есть:

- экран обзора коллекций;
- первый сценарий для каталога колокольчиков;
- поиск и фильтрация по состоянию;
- карточка предмета с характеристиками и заметками;
- архитектурная база, которую можно расширить на другие типы коллекций.

## Дальше можно добавить

- фотографии каждого предмета;
- редактирование и добавление новых экземпляров;
- локальное хранение через SwiftData;
- отдельные разделы для других коллекций.
