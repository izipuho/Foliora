# Privacy Policy for Foliora Bells

_Last updated: June 2026_

Foliora Bells is a personal collection catalog application.

The application stores collection data locally on the user's device. No personal information is collected, sold, or shared with third parties.

The application does not use advertising networks and does not track users across apps or websites.

Photos added to the catalog remain under the user's control and are stored on the user's device unless the user explicitly enables a cloud synchronization feature provided by Apple.

The application may use standard Apple services such as iCloud, if enabled by the user, solely for synchronization and backup purposes.

No account registration is required to use the application.
