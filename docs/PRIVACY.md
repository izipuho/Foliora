# Privacy Policy for Foliora Bells

_Last updated: July 2026_

Foliora Bells is a personal collection catalog application for storing information about bell collections, including collection names, item details, notes, storage locations, photos, documents, and 3D models that the user chooses to add.

## Data Collection

Foliora Bells does not collect, sell, or share personal information with third parties for advertising, analytics, or tracking purposes. The application does not use advertising networks and does not track users across apps or websites.

## Data Storage

Collection data is stored on the user's device. Media files and catalog details remain under the user's control unless the user enables Apple services such as iCloud synchronization, iCloud backup, or CloudKit sharing.

When iCloud synchronization or sharing is enabled, Apple may process catalog data, media, and sharing participant information according to the user's Apple ID, iCloud, and CloudKit settings. Foliora Bells uses these Apple services only to synchronize, back up, and share the user's catalog data as requested by the user.

## Photos, Camera, and Photo Library

Foliora Bells may request access to the camera or photo library so the user can attach photos to catalog items. Photos are used only for catalog features selected by the user, including local photo analysis that suggests item details from images. Photo analysis runs on the device and is not used for advertising or cross-app tracking.

## Import and Export

Foliora Bells may allow users to import or export catalog archives. Exported files are created only when requested by the user and can contain catalog data and media selected for export. Users are responsible for how they store or share exported files outside the application.

## Account Registration

No separate Foliora Bells account registration is required to use the application. Apple ID or iCloud account features are managed by Apple when the user chooses to enable Apple synchronization, backup, or sharing services.

## Data Deletion

Users can delete catalog content in the application. If iCloud synchronization is enabled, deletions may synchronize through iCloud for the same Apple ID according to Apple's service behavior and the user's iCloud settings.
